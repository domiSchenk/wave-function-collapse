# Wave function collapse algorithm 

This is a simple algorithm for wave function collapse.

it uses mulberry32 for random number generation. (so it can be seeded)  
if no seed is provided, it will use math.random.

## Starting

> $ npm install  
> $ npm run ts:start


## Parameters

    Height : number  
    Width : number
    Rules: Array<Rule>
    [debug: boolean]
    [seed: number ]
